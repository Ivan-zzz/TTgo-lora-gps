# TTGO-Lora-GPS
TTGO display + Lora + GPS + Compass

TTGO t-display module with 2 buttons
1 button = menu, 2nd button = select
GPS signal received (coordinates, altitude, cardinal, bearing, date, time)
Exchange of telemetry with the second device with the same firmware and coding indexes.
Select target coordinates (unit 2 or home position - can be adjusted)
Count distance to the target (mathematically counted via 2 points on the surface of the Earth)
light Cryptography to provide confidence of transferred data
Additional commands possible.

just for fun device, for camping, tourism etc.
